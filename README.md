# Real-Time Traffic Sign Recognition System

This project is a computer vision based Traffic Sign Recognition System developed using Python, TensorFlow/Keras, and OpenCV. The system classifies traffic signs such as Stop, No Entry, Speed Breaker, Hospital, Petrol Pump, and No Overtaking.

## Features
- Custom traffic sign dataset
- Image augmentation for improving dataset size
- CNN model training using TensorFlow/Keras
- Image-based prediction
- Model evaluation using accuracy and confusion matrix
- Scope for real-time webcam-based detection

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Dataset collection
2. Image preprocessing
3. Data augmentation
4. CNN model training
5. Model testing and evaluation
6. Prediction on new traffic sign images

## Classes Used
- Stop
- No Entry
- Speed Breaker
- Hospital
- Petrol Pump
- No Overtaking

## Result
The model achieved good training and validation accuracy on the custom dataset. Misclassifications were analyzed using a confusion matrix.
